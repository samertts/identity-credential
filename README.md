# identity-credential
**UICMP (Universal Identity &amp; Credential Management Platform)** هو نظام مكتبي احترافي يعمل بالكامل دون اتصال بالإنترنت لإدارة وإصدار الهويات والبطاقات التعريفية. يدعم تصميم القوالب، إدارة الأشخاص والصور، QR، الطباعة الجماعية، التدقيق والاعتماد، الأرشفة، النسخ الاحتياطي، وسجل العمليات، مع بنية مرنة قابلة للتوسع والتطوير للمؤسسات الحكومية والخاصة.
